About

StationFinder is a self-contained radio station database application. Load a schedule file (Aoki, EiBi, swskeds or s1b format) and search by frequency. Then filter by station name, mode, language, source, or on-air status. Optionally connect to SDRconnect via WebSocket to track and tune stations in real time.

Buttons

Import FileLoad a Aoki, EiBi, swskeds or s1b schedule file. ManualSearch manually by typing a frequency. FetchOne-shot read of the SDR VFO frequency. TrackContinuously follow the SDR VFO frequency. Reset AllClear all search fields and filters. Web Search 1Search current VFO frequency on shortwave.live. Web Search 2Search current VFO frequency on short-wave.info. Exact / Prefix / RangeSwitch frequency search mode. Range accepts e.g. 9000-10000 kHz. Show All / On NowFilter to stations currently on air based on UTC time. Language / SourceFilter results by language or schedule source. Double-click rowTune the SDR receiver to that station's frequency and mode.

Keyboard Shortcuts

F or fJump to frequency search box. S or sJump to station search box. EnterRun search and release focus. EscapeRelease focus from search box.
