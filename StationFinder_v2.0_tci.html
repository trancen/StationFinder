About

StationFinder is a self-contained radio station database application. Load a station frequency schedule file and search by frequency. Then filter by station name, mode, language, source, or on-air status. Optionally connect to a SDR program via WebSocket or CAT to track and tune stations in real time.

Buttons

Import FileLoad an Aoki, EiBi, HFCC, SDR Console, swskeds or s1b schedule file. ManualSearch manually by typing a frequency. FetchOne-shot read of the SDR VFO frequency. TrackContinuously follow the SDR VFO frequency. Reset AllClear all search fields and filters. Web Search 1Search current frequency on shortwave.live. Web Search 2Search current frequency on short-wave.info. Exact / Prefix / RangeSwitch frequency search mode. Range accepts e.g. 9000-10000 kHz. Show All / On NowFilter to stations currently on air based on UTC time. ModeFilter stations based on mode (AM, USB, CW etc.) Language / SourceFilter results by language or schedule source. Double-click rowTune the SDR receiver to that station's frequency and mode. Ctrl-click rowShow all fields for that station in a detail panel.

Keyboard Shortcuts

F or fJump to frequency search box. S or sJump to station search box. EnterRun search and release focus. EscapeClear and release focus from search box.

Importing Frequency List Files

StationFinder loads frequency lists in several formats. The easiest way to retrieve, save, and merge your frequency list files is with the List Builder, included in this package. Open it by clicking the link below or opening ListBuilder_v1.0.html.

Unsupported Browser

StationFinder requires a Chromium-based browser such as Google Chrome or Microsoft Edge.

Features such as COM port access (CAT / CI-V) and file handling require APIs that are not available in your current browser.
