📡 ListBuilder v1.0

ListBuilder is a program to retrieve, convert, save and merge frequency lists from multiple sources into a format compatible with StationFinder. For each frequency list source, click Get file for download and save instructions. If you want to merge several sources into one file Import the desired files and then click Merge & Save.

📡 Getting the Aoki schedule file

1
Visit the Aoki database page:
https://www1.s2.starcat.ne.jp/ndxc
2
Download the current season zip file (e.g. a26, b26 etc.).
3
Save the zip file — do not extract it. StationFinder and ListBuilder read zip files.
The zip contains the schedule data file in several formats. StationFinder and ListBuilder automatically use the correct one.

📡 Getting the EiBi schedule file

1
Visit the EiBi database page:
http://www.eibispace.de
2
Download the latest CSV database file. It will be named something like sked-a26.csv.
3
Save the file — no extraction needed. It is in a compatible format for StationFinder.
The filename must start with sked- and have a .csv extension.

📡 Getting the HFCC schedule file

1
Visit the HFCC public data page:
http://www.hfcc.org/data/
2
Download the current season zip file, e.g. a26allx2.zip (B = winter, A = summer, followed by the two-digit year).
3
Save the zip file — do not extract it.
The zip contains multiple files that need to be converted to a single file in a compatible format.
4
Click Convert to process the zip file, then Save to write the file for use with StationFinder.

📡 Getting the ILGRadio schedule file

1
Visit the ILGRadio download page:
https://www.ilgradio.com/Download/databases/A-B-database.html
2
Download the two zip files in CSV format:
ilgaacsv.zip — Broadcasting and Utility, full 1.6–30 MHz coverage.
ilgbbcsv.zip — Broadcasting, Weather, VOLMET and Time Signal stations only.
3
Save the zip files — do not extract them. They need to be converted to a compatible file format.
Note: ILGRadio is a paid database. A subscription is required to download the files.
4
Click Convert to process the desired zip file, then Save to write the file for use with StationFinder.

📡 Custom file format

The custom file must be a comma-delimited CSV with a header row containing these exact column names in this order:

Frequency, M, Station, On, Off, Language, Site, TX Country, Days, Target, Power, Azimuth, Origin, Source
Notes:
• Frequency must be in Hz (e.g. 9500000 for 9500 kHz)
• M is the modulation mode (AM, USB, CW etc.)
• On / Off times in HHMM format (e.g. 0600, 1800)
• Source will be tagged as Custom if left blank
• Missing columns are left empty — not all columns are required
The easiest way to create a compatible file is to export a merged file from ListBuilder and use that as your starting point.

📡 Getting the swskeds schedule file

1
Visit the swskeds group (membership required):
https://groups.io/g/swskeds
2
Go to the Files section and download the current season zip file.
3
Save the zip file — do not extract it. StationFinder and ListBuilder read zip files.
swskeds is a combined Aoki/EiBi/HFCC database maintained by the swskeds group. Membership is required to access the Files section.

Unsupported Browser

ListBuilder requires a Chromium-based browser such as Google Chrome or Microsoft Edge.

Features such as file picking and saving require APIs that are not available in your current browser.
